# Execution Intent (Phase 5K)

Institutional purpose and directional outcomes across EXECUTIA surfaces — presentation only. No API, SQL, or execution logic changes.

## Objective

Execution must feel:

- purpose-driven
- directionally governed
- outcome-oriented
- institutionally intentional
- execution-aware

Not like a workflow engine, process tooling, operational dashboard, or task orchestration system.

## Intent layer

| Asset | Role |
|-------|------|
| `executia-execution-intent.js` | Objective framing, intent band, `applyIntent` |
| `executia-execution-intent.css` | Institutional intent band + objective blocks |

Imported via `executia-governed-presentation.css`. Enable with `ex-intent-enabled`.

## Canonical language

| Frame | Use |
|-------|-----|
| EXECUTION OBJECTIVE | Primary institutional purpose |
| GOVERNED OUTCOME | Outcome posture from readiness/risk |
| CANONICAL INTENT | Canonical layer alignment |
| EXECUTION DIRECTION | Directional movement toward objective |
| INSTITUTIONAL OBJECTIVE | Executive/audit objective continuity |
| DETERMINISTIC PURPOSE | Mission continuity across transitions |

Avoid: tasks, workflow, productivity, project-management wording.

## Intent hierarchy

| Tier | Role | Class |
|------|------|-------|
| Primary | Execution objective | `.ex-intent-primary` |
| Secondary | Governance continuity | `.ex-intent-secondary` |
| Tertiary | Diagnostics | `.ex-intent-tertiary` |

## Engineering console

Intent band mounts below memory; state derived from execution intelligence (readiness, risk, recommendations).

```javascript
INTENT().applyIntent(ctx);
```

Surfaces use objective framing:

```javascript
INTENT().objectiveFramingHtml(ctx);
INTENT().governedOutcomeLabel();
```

## Descriptors

- Execution exists to advance governed outcomes — institutional direction precedes operational motion.
- Execution moves toward canonical intent; consequence and memory preserve objective continuity.
- Governed outcomes remain deterministic — purpose binds authority, replay, and verify.
- Deterministic purpose sustains mission continuity across governance transitions.

## Preserved

Execution memory (5J), consequence (5I), rhythm (5H), governance core (5G), institutional calm, deterministic atmosphere.
