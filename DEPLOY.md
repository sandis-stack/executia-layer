# Deploy

1. Copy all files into `sandis-stack/executia-layer`.
2. Run `npm install`.
3. Run `npm run verify`.
4. Commit and push.
5. Redeploy Vercel.

Routes:
- `/dashboard`
- `/control`
- `/history`
- `/operations`
- `/demo`
- `/api/v1/health`
- `/api/v1/execute`
- `/api/v1/operator-decision`
- `/api/v1/ledger-verify`
