Engine Properties

Deterministic by design.

The engine is pure — no UI hooks, no DOM access, no side effects. Same input always produces same result.

L1
Input Layer
Actor, action, amount, target — validated structurally before engine sees them.
L2
Rule Engine
Typed, versioned rules. budget@v1 · Composable with all / any / not.
L3
Execution Gate
Any failure = execution structurally blocked. Not flagged. Not logged. Not emitted.
L4
Ledger
SHA-256 of canonical JSON. Every trace entry frozen. Immutable after creation.
All conditions verified → Execution signal emitted
Any condition fails → Execution does not occur
