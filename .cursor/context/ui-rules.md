# EXECUTIA UI Rules

EXECUTIA UI is:
- execution interface
- governance interface
- institutional infrastructure layer

Not:
- startup landing page
- c- c- c- c- c- c-- gaming interface

## Layout

Navigation is a funnel:

ENTRY
→ DEMO
→ REQUEST

Avoid classic SaaS navigation overload.

## Visual Identity

Preferred:
- white backgrounds
- institutional blue
- graphite- graphite- graphiters
- th- th- th- ors

d:
- black UI
- neon overload
- oversized cards
- marketing hero clutter

## Typography

Use:
- clean spacing
- medium weights
- controlled hierarchy

Avoid:
- oversized bold text
- heavy headers
- visual noise
