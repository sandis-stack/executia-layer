# Institutional multi-surface (Cursor)

EXECUTIA = **one infrastructure**, **eight surfaces**. Do not collapse into a single ENGINE page.

Registry: `public/components/executia-institutional-surfaces.js`

Surfaces: Execution (engine home), Governance, Proof, Replay, Health, Operations, Engineering, Request.

Public pages: load `executia-institutional-surfaces.js` before `executia-institutional-environment.js`.

Console pages: load surfaces before `executia-governance-core.js` so NAV stays aligned.
